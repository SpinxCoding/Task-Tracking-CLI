# Task Tracker CLI
A simple command line interface as a task manager | JSON based.

---

## Features
- Add tasks
- Update task descriptions
- Delete tasks
- Mark tasks as 'in-progress' or 'done'
- Able to list all tasks, with custom arguments ('todo', 'done', 'in-progress'

---

## Requirements
1. Python 3
2. git
